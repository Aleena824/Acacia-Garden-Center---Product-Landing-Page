# Acacia Garden Center - Product Landing Page
This project is a responsive product landing page for **Acacia Garden Center**, showcasing a popular indoor plant - the **Swiss Cheese Plant (Monstera deliciosa)**.

The website is designed to simulate a real e-commerce product page with a focus on clean UI, user experience, and modern web design practices.

## Features

* **Navigation Bar**

  * Smooth in-page navigation (Home, About, Contact)

* **Hero Section**

  * Eye-catching background with overlay
  * Clear headline and call-to-action

* **Product Section**

  * Product image and details
  * Pricing and product specifications

* **Product Details**

  * Description of the plant
  * Key features displayed in a grid layout
  * Suitable usage areas

* **Why Buy Section**

  * Highlights benefits of the product
  * Interactive hover cards

* **Call-To-Action**

  * “Add to Cart” button with alert interaction

* **Contact Form**

  * User input for name, email, and message

* **Responsive Design**

  * Optimized for mobile and desktop screens

## Technologies Used

* **HTML5** – Structure of the webpage
* **CSS3** – Styling, layout, and responsiveness
* **JavaScript** – Basic interactivity (alert on button click)
